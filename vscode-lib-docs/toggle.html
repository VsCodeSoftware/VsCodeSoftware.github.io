Toggle

Booting Window Toggle Slider Dropdown

Tab:CreateToggle(options)

Tạo một nút bật/tắt có thể lưu trạng thái.

local Toggle = Tab:CreateToggle({
    Name = "Speed Hack",
    CurrentValue = false,
    Flag = "SpeedHack",            -- Bắt buộc nếu dùng lưu config
    Callback = function(Value)
        game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = Value and 100 or 16
    end
})

Bảng tham số

Tham sốKiểuMô tảBắt buộc
NamestringTên hiển thị của toggleYes
CurrentValuebooleanGiá trị mặc định (true/false)No (default false)
FlagstringTên để lưu config (phải duy nhất)Yes nếu bật ConfigurationSaving
CallbackfunctionHàm chạy khi thay đổi giá trịYes
Lưu ý: Nếu bạn bật ConfigurationSaving trong CreateWindow, mọi toggle có Flag sẽ tự động lưu & load lại khi vào game.
