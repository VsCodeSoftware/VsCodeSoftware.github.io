Booting the Library

Chỉ cần 1 dòng để load thư viện từ GitHub:

local VSCodeLib = loadstring(game:HttpGet("https://raw.githubusercontent.com/username/vscode-lib/main/lib.lua"))()
Tip: Đặt đoạn code này ở đầu LocalScript (StarterPlayer → StarterPlayerScripts).

Create Window

local Window = VSCodeLib:CreateWindow({
    Name = "My Script Hub",
    LoadingTitle = "VSCodeLib",
    LoadingSubtitle = "by @username",
    ConfigurationSaving = {
        Enabled = true,
        FolderName = "VSCodeLibConfig",
        FileName = "Settings"
    },
    Discord = {
        Enabled = true,
        Invite = "dsc.gg/yourserver",
        RememberJoins = true
    },
    KeySystem = true,
    Key = "ABC123XYZ"
})

Window:Notify({
    Title = "Welcome!",
    Content = "VSCodeLib đã load thành công!",
    Duration = 5
})

Các thành phần UI hỗ trợ

Tabs → Sections → Button, Toggle, Slider, Dropdown, Keybind, Colorpicker, Textbox...

-- Ví dụ Toggle
local Toggle = Tab:CreateToggle({
    Name = "Speed Hack",
    CurrentValue = false,
    Flag = "SpeedToggle",
    Callback = function(Value)
        game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = Value and 100 or 16
    end,
})

Configuration Saving

Tự động lưu tất cả flag khi thoát game và load lại khi vào.

Quan trọng: Mỗi Toggle/Slider/Dropdown phải có Flag = "Tên duy nhất" để lưu được.
